Relações e Facetas

Soluções interativas para correlações, tendências (geom_smooth) e subgráficos (facets).

Exercício 1

Tendências por Categoria

library(tidyverse) ggplot(mpg, aes(x = displ, y = hwy, color = drv)) + geom_point(alpha = 0.5) + geom_smooth(method = "lm", se = FALSE) + labs(title = "Tendência de Consumo por Tipo de Tração")

Nota Pedagógica: Ao colocar color = drv DENTRO da função principal ggplot(aes(...)), a função geom_smooth herda esse agrupamento e entende automaticamente que deve calcular uma linha de regressão separada para cada grupo de tração.

Exercício 2

O Poder das Facetas (Scales Free)

ggplot(mpg, aes(x = displ, y = hwy)) + geom_point() + facet_wrap(~ class, scales = "free")
Controlo de Escala:
Resposta: Ao usar scales = "free", cada subgráfico define os seus próprios limites de X e Y. Isso é bom para ver o padrão individual, mas péssimo para comparar classes entre si, pois perde-se o referencial absoluto.
Exercício 3

Cruzamento de Variáveis (Facet Grid)

ggplot(mpg, aes(x = displ, y = cty)) + geom_point() + facet_grid(drv ~ cyl)

Observação: Sim, existem células vazias (ex: tração traseira r com 4 cilindros). Isto indica que não existem carros com essa configuração específica no dataset. O facet_grid é excelente para detetar estas ausências visualmente.

Exercício 4

Combinando Tudo

ggplot(mpg, aes(x = cty, y = hwy, color = class)) + geom_point() + geom_smooth(color = "black", size = 0.5) + facet_wrap(~ year) + theme_bw()

Este gráfico final combina mapeamento estético por cor (class), geometrias múltiplas (pontos e linha de tendência global preta) e subdivisão espacial por facetas (year), resultando numa visualização altamente densa em informação, mas legível graças ao theme_bw().
