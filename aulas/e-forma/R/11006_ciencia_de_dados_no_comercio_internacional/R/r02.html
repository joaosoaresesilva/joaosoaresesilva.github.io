Guia de Resolução: Aula 02

ggplot2 & Distribuições

Respostas aos Exercícios Propostos

Este guia contém o código R necessário para resolver os desafios da aula02.html, com foco em manipulação de variáveis categóricas e ajuste de visualizações de distribuição.

1

Frequência de Fabricantes

O objetivo é mostrar a contagem de modelos por fabricante, ordenados por frequência. Usamos fct_infreq para que as barras fiquem organizadas visualmente.

# Carregar bibliotecas necessárias
library(tidyverse)

# Criar o gráfico com ordenação automática
ggplot(data = mpg) +
  geom_bar(mapping = aes(x = fct_infreq(manufacturer), fill = manufacturer)) +
  coord_flip() + # Inverter para facilitar leitura de nomes longos
  theme_minimal() +
  labs(x = "Fabricante", y = "Nº de Modelos", title = "Modelos por Fabricante")
2

O Impacto dos Bins

Ajustar o número de compartimentos (bins) altera o detalhe da distribuição. Com 10 bins vemos a forma geral; com 50 vemos as lacunas nos dados.

# 10 Bins (Mais Generalista)
ggplot(mpg, aes(x = displ)) +
  geom_histogram(bins = 10, fill = "blue")
# 50 Bins (Mais Detalhado)
ggplot(mpg, aes(x = displ)) +
  geom_histogram(bins = 50, fill = "red")

Resposta: Geralmente, um valor intermédio (como 30) é melhor. O de 50 bins revela que a cilindrada não é perfeitamente contínua, mas sim agrupada em valores padrão de motorização.

3

Consumo por Cilindrada (Densidade)

Como cyl é lida como número, temos de transformá-la em fator para que o ggplot atribua cores diferentes a cada categoria discreta.

ggplot(data = mpg, mapping = aes(x = cty, color = factor(cyl))) +
  geom_density(size = 1) +
  labs(color = "Cilindros", x = "Consumo Cidade (MPG)")
4

Identificar Outliers com Boxplots

O boxplot é a ferramenta ideal para ver a dispersão e identificar pontos que fogem à norma (outliers).

ggplot(data = mpg, aes(x = drv, y = hwy, fill = drv)) +
  geom_boxplot() +
  theme_light() +
  scale_fill_brewer(palette = "Set2")

Dica: Os pontos pretos acima ou abaixo das "antenas" do gráfico representam os outliers.

5

Design: Transparência e Sobreposição

Usar alpha é fundamental quando temos várias distribuições no mesmo eixo. Removemos a legenda conforme solicitado no exercício.

ggplot(mpg, aes(x = hwy, fill = drv)) +
  geom_density(alpha = 0.4) +
  theme(legend.position = "none") +
  labs(title = "Distribuição de Consumo por Tração (Sem Legenda)")
