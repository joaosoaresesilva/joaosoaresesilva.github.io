Soluções Interativas (ggplot2)

Este documento interativo contém as soluções e simulações visuais para os exercícios propostos na aula de introdução à Gramática de Gráficos. Explore o código e interaja com os gráficos para consolidar a sua aprendizagem.

Exercício 1

Exploração Inicial

Objetivo: Criar um gráfico de dispersão simples.

library(tidyverse) # Gráfico de dispersão entre consumo na cidade (cty) e autoestrada (hwy) ggplot(data = mpg, mapping = aes(x = cty, y = hwy)) + geom_point()

Observação do Formador:

Nota-se uma correlação linear positiva forte. À medida que o consumo na cidade aumenta, o consumo na autoestrada também aumenta, o que é logicamente esperado.

Exercício 2

Mapeamento de Cores e Formas

Objetivo: Compreender mapeamento múltiplo e restrições de escala.

ggplot(data = mpg, mapping = aes(x = cty, y = hwy, color = drv, shape = drv)) + geom_point(size = 3)

Resposta à Questão Teórica:

O que acontece se mapear uma variável contínua (ex: displ) à estética shape?

  • Resultado: O R retornará um erro: "A continuous variable can not be mapped to shape".
  • Porquê: Formas geométricas (círculo, triângulo, quadrado) são categorias discretas. Não existe uma "forma intermédia" entre um círculo e um triângulo que represente, por exemplo, o valor 2.5.
Exercício 3

Mapeamento Estático vs Dinâmico

Objetivo: Distinguir entre "definir um valor" (estático) e "mapear uma variável" (dinâmico). Utilize os botões abaixo para simular as duas abordagens e observe o código alterado.

# A cor 'blue' está FORA do aes() porque não depende dos dados ggplot(data = mpg, mapping = aes(x = displ, y = hwy)) + geom_point(color = "blue")
Exercício 4

Introdução ao geom_smooth

Objetivo: Trabalhar com múltiplas geometrias e demonstrar a independência das camadas. Alterne as camadas abaixo para ver o resultado.

# Com pontos e linha de tendência ggplot(data = mpg, mapping = aes(x = displ, y = hwy)) + geom_point() + geom_smooth()

O que acontece ao remover os pontos? O gráfico exibe apenas a linha de tendência. Isto demonstra que as camadas são independentes, mas partilham os dados e eixos definidos na base ggplot().

Desafio Final

Personalização (Exercício 5)

Objetivo: Aplicar estética, temas e labels profissionais. Este é o resultado que se espera após a combinação de todas as camadas de formatação.

ggplot(data = mpg, mapping = aes(x = displ, y = hwy, color = factor(cyl))) + geom_point(size = 3, alpha = 0.8) + geom_smooth(se = FALSE, color = "black", linetype = "dashed") + labs( title = "Relação Consumo vs. Motorização", subtitle = "Análise por número de cilindros", x = "Cilindrada do Motor (L)", y = "Consumo em Autoestrada (MPG)", color = "Cilindros", caption = "Elaborado para a Formação de R" ) + theme_light() + theme( plot.title = element_text(face = "bold", size = 14), legend.position = "bottom" )

Dica Pedagógica:

Reforce com os alunos que o factor(cyl) é usado no desafio porque cyl é lido como número, mas para fins de cor, queremos tratá-lo como categoria (4, 6 ou 8 cilindros).
